# DraftPilot — Frontend

Web UI for configuring and monitoring DraftPilot drafts.

The frontend is used to set up the worker and monitor draft progress via the backend API.

➡️ **Project overview:** https://github.com/kevinperumal/draftpilot

## Responsibilities
- Configure and start draft monitoring
- Display draft state and progress
- Communicate with the backend API

## Tech stack
TypeScript, React

## Notes
This UI does not contain secrets; all sensitive configuration is handled by the backend and worker.
